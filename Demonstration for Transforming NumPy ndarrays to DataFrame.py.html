Demonstration for Transforming NumPy ndarrays to DataFrame.py 
import numpy as np
import pandas as pd

# We suppose that we have the following NumPy array with stock prices :
price_array = np.array([100, 101, 102])

# We suppose another NupPy array with volumes :
volume_array = np.array([200, 210, 220])

# We can create a DataFrame like to :
df_from_array = pd.DataFrame({
    "Price": price_array,
    "Volumes": volume_array
})
print(df_from_array)
